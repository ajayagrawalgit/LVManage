What is LVManage ?
It is a command line utility which will help you to manage all your LVM needs.

Do I need any specific license t use this ?
No! The tool is completely free to use and you can even download the source code and modify it as per your use.

Do I need a prior knowledge of LVM for using LVManage ?
No. If you just know what is LVM and understand some basic concepts about it. You're good to go.

Will I have to be familiar with the commands and syntaxes for any of the activity related to LVM in order to use this ?
No. You'll be guided throughout. Don't worry. You'll be fine :)


---------------------------------------------------------------------------
---------------------------------------------------------------------------

How to use this tool ?
- Place the entire directory in your linux machine (location doesn't really matter).
- Open the LVManage directory.
- Run the script called LVManage.sh (using sudo is recommended ->> sudo ./LVManage.sh).
